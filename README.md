# H.bilibili-Angular
一个基于 angular & ionic mobile app 

## 开始
* 安装node环境
* 克隆这个仓库: `git clone https://github.com/Feng-Bu-Jue/H.bilibili-Angular.git`.
* 安装angular和ionic的cli `npm install -g @angular/cli` `npm install -g ionic`
* `npm install`
* `npm run build:lib` 
* `npm run start`
